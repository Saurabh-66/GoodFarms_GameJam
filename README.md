A farming simulation game built in Python and Pygame (submission for the Game Jam under the theme Growth and Decay)

## About the Game
GoodFarms is a 2D top-down farming game where everything lives and dies on its own clock. You till the land, sow seeds, water your crops and watch them grow. The world around you shifts between day and night, rain comes and goes without warning and the seasons of each day remind you that nothing stays still for long.

## Features

Farming system - Till the soil, plant seeds, water them and watch corn and tomatoes grow over time

Apple trees - Trees fruit naturally and you can collect the apples and cut trees using the axe

Day and night cycle - The world transitions smoothly from morning to night, with a sleep mechanic to advance the day

Dynamic weather - Rain falls randomly and waters your crops automatically but you never know when it's coming

Merchant trading - Sell your harvest and buy crops from the trader to keep the farm running

Original background music - Composed and produced from scratch specifically for this game (using Audacity)

Tile-based world - Built using Tiled map editor

## How to Run

**Requirements:**

```
Python 3.6 or higher
pygame
pytmx
```

**Install dependencies:**

```
pip install pygame pytmx
cd code
python main.py
```
